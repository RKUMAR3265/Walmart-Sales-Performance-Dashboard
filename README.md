# Walmart-Sales-Performance-Dashboard
## Project Overview
This project focuses on building an interactive **Walmart Sales Performance Dashboard** using **Power BI**.
The dashboard provides executive-level insights into sales revenue, profit, orders, customer segments, and top-performing cities and products.
The objective of this project is to convert raw retail sales data into meaningful business insights using data analytics and visualization techniques.

## Objectives
- Analyze Walmart sales performance using KPIs
- Identify monthly sales trends
- Compare sales and profit performance
- Identify top-performing cities and products
- Understand customer segment contribution
- Build an interactive and professional dashboard
